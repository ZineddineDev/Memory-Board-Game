# Memory Board Game 🎮

An interactive C++ console-based memory game that tests the player's focus and pattern recognition. 
The game challenges players to find and match hidden values within a grid,
featuring a dynamic feedback system.

###🚀 Features (الميزات)

1. Play with computer
2. Points system  ( You can play and sum 'Point' )
3. Intercative interface  ( Massage appeared And colors changed )
4. Error management ( Ensuring that the user enters only correct options )

#### 💻 Technologies ( التقنيات المستخدمة )

Languages : C++
IDE : Visual Studio , ( You can use any ide for trying )
Logic : loops,condition statements(if,if else ,switch),Functions,Enem,Structers.

#####🛠️ How to play

Get the code in file (Project.cpp) and doing run in any IDE
